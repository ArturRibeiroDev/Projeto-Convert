# 🔄 Convert Template

Template interativo em JavaScript vanilla para manipulação de dados e exibição dinâmica no DOM.

## 🚀 Tecnologias

| Tecnologia | Função |
|---|---|
| **JavaScript** | Manipulação do DOM, eventos |
| **HTML5** | Estrutura semântica |
| **CSS3** | Estilização |

Abra `index.html` no navegador. Não requer servidor.

## 💡 O que foi aprendido

- Conversão e manipulação de dados no frontend
- Event handling e atualização dinâmica do DOM
